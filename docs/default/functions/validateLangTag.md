# Function: validateLangTag()

```ts
function validateLangTag(lang): boolean;
```

## Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `lang` | `string` | a language tag |

## Returns

`boolean`

Returns `true` if the language tag is valid, else `false`.
